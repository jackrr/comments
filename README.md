# Comments

This is a blog (barely).

## Your task

Add the ability to comment on a blog post.

This prompt is intentionally open-ended. You will likely not be able
to fully build out a full-stack comment experience within the next 45
minutes and that's fully OK.

## Dev setup

Install dependencies:
```
npm install
```


Run the migrations/bootstrap your db:
```
npx prisma migrate dev
```

Run the next.js server:
```
npm run dev
```

In a separate terminal, run the express server REST API:
```
npm run server
```

Visit `http://localhost:3000`
